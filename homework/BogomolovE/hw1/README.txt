В этом проекте реализована программы по подсчету суммы элементов двумерного массива. 
Сумма вычисляется двумя способами:
1) сложение производится по элементам строк (реализовано в файле sum_by_cols.cpp);
2) сложение производится по элементам столбцов (реализовано в файле sum_by_rows.cpp).

Сделано это для того, чтобы показать различие в эффективности работы при разном способе индексирования по массиву.
Для этого используется специальный класс Timer, отражающий время работы участка кода.

Детали реализации:

1) при помощи функции malloc() выделяется необходимая место для хранения массива в динамической памяти;
2) используя псевдоним типа (using) указатель на созданную malloc() память преобразуется к типу (int *)[N][M]
3) освобождается память при помощи функции free()
