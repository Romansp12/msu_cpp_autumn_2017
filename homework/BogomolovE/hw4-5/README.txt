В этом проекте реализован класс Matrix и его тестирование. 

Реализовано и перегружено:

1) конструкторы: инициализации, копирования, перемещения;
2) операторы: копирования, перемещения;
3) оператор получения элемента перегружен с помощью дополнительного класса Row, позволяющего представить в виде объекта строку матрицы;
4) методы: вывода, получения количество строк и столбцов;
5) операторы умножения на std::vector, на число слева и справа перегружены, как friend  методы
6) операторы сравнения на равенство (==) и неравенство перегружены (!=), как friend методы класса

Класс Matrix может быть создан от любого встроенного типа благодаря использованию template
