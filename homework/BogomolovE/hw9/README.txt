В этом проекте реализована программа, ведущая подсчет одинаковых слов в input файле, и выводящая результат в порядке убывания частот в output файл

Особенности реализации:

1) Для подсчета слов используется ассоативный массив unordered_map, храянящий в себе {слово: частоту этого слова}

2) Для сортировки значений частот введен асcоциативный массив multimap, хранящий в себе {частота слова: итератор пары (слово:частота) в массиве unordered_map}.
Последняя структура необходима для установления порядка в unordered_map, в ней лежат пары отсортированные по ключу в порядке возрастания. 
Вместо map используется multimap с целью сохранения всех одинаковых ключей.

3) Итерируясь с конца multimap выводим в выходной файл слова и их частоты. Эта функция работает ожидаемым образом, если в той области видимости, где она была вызвана существует
unordered_map, итераторы которого записаны в multimap.
