Программа составляет частотный словарь "слово" - "сколько раз встречается".
Программе на вход через аргументы командной строки передается два имени файлов. 
Первое имя - текстовый файл, в котором слова разделены пробелами, второе имя - текстовый файл, 
в который выводится результат.
Словарь отсортирован по частоте.
