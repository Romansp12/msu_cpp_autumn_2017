Калькулятор обрабатывает выражения вида "2 + 3 * 4 - -2" методом рекурсивного спуска. 

Если выражение содержит символы, отличные от пробелов, цифр, знаков операций, результатом 
выполнения программы будет сообщение "Invalid symbol in expression!"

Если в выражении два числа разделены только пробелом, результатом будет сообщение 
"Wrong expression!"

Поддерживаются операции умножения, деления, сложения, вычитания и унарного минуса.

В случае деления на ноль результат - "Division by zero!".
