Реализация класса содержится в файле Matrix.cpp
Реализация тестирования матрицы в файле test.cpp

После запуска make запускать файл test

Поддерживаются операции:
  Получить число строк ( столбцов ) - метод rows() ( columns() ) 
  Получить конкретный элемент - [][]
  Умножить на вектор -  m *= v, для v проверяется равенство числа элементов количеству строк в матрице
  Умножить на число - m *= k
  Сравнение матриц на равенство - ==, истинный ответ возможен только при совпадении размерностей матриц
  Сравнение матриц на неравенство - != - противоположность ==
  Вывод матрицы  - метод print() - принимает аргумент: 
    int with_size - флаг вывода размерностей матрицы перед выводом
                    ее элементов на отдельной строке
    char *format - опциональный параметр - форматная строка для вывода одного элемента (при отсутствии используется "%lf ")
    Между строками матрицы производится один перевод строки
