## Калькулятор арифметических выражений

Утилита командной строки, вычисляющая значение арифметического выражения, состоящего из целых чисел.

Поддерживаемые константы

    e
    Pi

### Компиляция

Из директории исходного файла выполнить

    make

### Примеры использования

    ./calc "-12 * 2 / 3 + 8 * 2 / 1"
    ./calc "(5 + 3 * (8 / Pi) * 4) + (1)"
