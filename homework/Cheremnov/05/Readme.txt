Сборка программы производится с использованием утилиты make. (Makefile для этого находится в папке).
Работа программы:
Если в качестве второго аргумента командной строки передаётся:
-errorrow, тогда программа тестирует выход за границы строк матрицы
-errorcolumn, тогда программа тестирует выход за границы столбцов матрицы
-errorvector, тогда программа тестирует несовпадение размерностей при умножении.
В ином случае она запускает серию тестов для матрицы.
