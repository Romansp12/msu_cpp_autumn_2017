#Домашнее задание №2
Простой калькулятор. Написан с использованием метода рекурсивного спуска. Поддерживает:
- сложение
- деление
- умножение
- вычитание
- унарный минус

 Сборку производить с помощью утилиты make.
'''
	make
'''
Запуск осуществляется с помощью
'''
	./calc 2+3*4
'''
Вывод
'''
	14
'''
Грамматика калькулятора:
```
expr = term
    | expr + term
    | expr - term
term = prim
    | term * prim
    | term / prim
prim = number
    | -number
number = [0-9]+
```
