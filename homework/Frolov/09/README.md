#Домашнее задание №9. 
Программе через аргументы командной строки передают два имени файлов. Подсчитывает частоту слова в файле.

 Сборку производить с помощью утилиты make.
```
	make
```

##### Пример

```input.txt```

```
grass tree forest grass
```

```output.txt```

```
2 grass
1 tree
1 forest
```
