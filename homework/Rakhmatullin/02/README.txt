Для компиляции приложения калькулятор: make calc

Для запуска приложения: ./calc.exe "<your expression>"

Выражение можно вводить с пробелами. 

Примеры: 

"2 -- 2"
"2*3 -4"
"12*12/30"

Execptions: 
1) Zero divisions (деление на ноль)
2) Unexpected symbol (введен отличны символ от "+ - * / [0-9]", или некорректная последовательность символов типа "++")
