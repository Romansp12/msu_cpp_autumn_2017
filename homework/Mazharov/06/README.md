## Калькулятор

Простой калькулятор, реализованный с помощью метода рекурсивного спуска

Поддерживаются типы int, long, double
Осуществляется проверка на соответствие диапазону

### Компиляция

Выполнить

    make

### Запуск

    ./calc "expression"

### Запуск тестов

	./calc "test"

### Пример

	./calc "2 + 3 * 4 - -2"
